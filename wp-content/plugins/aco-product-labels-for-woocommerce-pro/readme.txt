== Description ==
WooCommerce Product labels helps to create product labels or product badges easily. This plugin provides an easy to use user interface to create custom product labels for woocommerce products. You can add labels to a set of selected products / categories or can replace default Woocommerce On Sale badges.

== INSTALLATION ==

Installing "Product Labels For Woocommerce" can be done either by searching for "Product Labels For Woocommerce" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
= Can I add badge on categories and tags? = 
Yes, you can do this by creating custom product lists. You can also set badges to all products on sale.

= Is it possible to set multiple badges? =
Yes, you can set multiple badges.
